# 📅 Glass Calendar App

A modern **glassmorphism-based calendar web app** with smooth UI, interactive features, and responsive design.

---

## ✨ Features

- 📆 Dynamic calendar (month & year navigation)
- ⏰ Live digital clock
- 🌙 Light / Dark theme toggle
- 🎯 Click on any date to view details
- 🧊 Glassmorphism UI design
- 🎨 Animated gradient background
- 📱 Fully responsive (mobile-friendly)
- 🧑‍💻 Minimal branding sidebar (Vinay Tiwari)
